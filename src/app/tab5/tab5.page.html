Frequently Asked Questions

How do I list and search for items?

  • Go to the Search page.
  • Enter the item name in the search field and click “Search”.
  • All matching records will appear in the table below.

How do I add new items and view featured items?

  • Navigate to the Add Item page to view a list of the featured items.
  • To add an item, fill in all required fields on the page (name, category, quantity, etc.).
  • Click “Submit” to add the new item.

How can I update or delete an existing item?

  • Open the Edit Items page.
  • Search for the item by name and click “Search”.
  • Use the “Edit” button to change details, then click “Update”.
  • Use the “Delete” button to remove the item.

Where can I find the privacy and security information?

  • Select the Privacy & Security page.
  • Read the guidelines on data handling, user permissions, and security best practices.

Troubleshooting Common Issues

I can't log into my account

  • Check that your username and password are correct.
  • If you forgot your password, use the "Forgot Password" link to reset it.

Item details aren't saving correctly

  • Ensure all required fields are filled out correctly.
  • Also, check for any special characters or invalid inputs.

The system is running slowly or pages aren't loading properly

  • Clear your browser cache, refresh the page, or try a different browser.
  • Check your internet connection..

Any other issues

  • Note down the error message and contact technical support.
